Introduction

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

What Does HTML Do?

HTML is used to:

  • Structure content with elements like headings, paragraphs, lists, images, links, and more.
  • Define the meaning and layout of web content.
  • Work together with CSS (for styling) and JavaScript (for interactivity).

Basic HTML Example

<!DOCTYPE html>
  <html>
     <head>
       <title>My First Webpage</title>
     </head>
        <body>
           <h1>Hello, World!</h1>
             <p>This is my first HTML page.</p>
        </body>
  </html>

Structure

Basic HTML Structure

HTML documents follow a specific structure that browsers understand. This structure ensures that content is well-organized and displayed correctly.

HTML Boilerplate Example


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" 
    content="width=device-width, initial-scale=1.0" />
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>>
  </body>
</html>

Breakdown Of Each Part

TagsPurpose
<!DOCTYPE html>Tells the browser this is an HTML5 document.
<html lang="en">Root of the document, <lang="en"> specifies the language.
<head>Contains meta info: title, encoding, etc. (not visible to users).
<meta charset="utf-8>Sets the character encoding (supports most characters).
<meta name="viewport" content="width=device-width, initial scale=1.0>Makes site mobile-responsivec
<title>Sets the page title shown on the browser tab.
<body>Contains visible content like text, images, buttons, etc.

CSS Basics

CSS (Cascading Style Sheets) is used to control the appearance and layout of HTML elements. It separates content (HTML) from design (CSS), allowing for more flexible and maintainable code.

What CSS Controls:

  • Fonts and text styles
  • Colors and backgrounds
  • Spacing (margins, padding)
  • Element positioning and layout
  • Animations and transitions
  • Responsive design

Ways To Apply CSS:

1. Inline - inside the HTML element using the <style> attribute

<p style="color: blue;">This is blue text.</p>

2. Internal - within a <style> tag inside the <head> of your HTML

           <style>
              p {
                Color: green;
              }
           </style>

3. External - using a separate .css file linked with <link>

<link rel="stylesheet" href="styles.css" />

CSS Syntax:

selector {

property: value;

.

}

Example:

h1 {

color: red;

font-size: 32px;

}

Common Selectors:

  • element - targets by tag (e.g. p, div)
  • .class - targets by class name (e.g. .box)
  • #id - targets by ID (e.g.#header)

Layout with Flexbox

Flexbox (Flexible Box Layout) is a CSS layout module designed to make it easier to design flexible and responsive layout structures without using floats or positioning hacks.

Why Use Flexbox?

Flexbox simplifies aligning elements, distributing space, and adapting to different screen sizes.

Enabling Flexbox:

Apply display: flex to a container to turn its children into flex items

.container {

display: flex;

}

Key Concepts:

1. Main Axis & Cross Axis

  • Main Axis: The direction items are laid out (default is horizontal).
  • Cross Axis: Perpendicular to the main axis

Flex Container Properties

1. flex-direction

Defines the direction of the main axis

Values:

  • row (default): Items arranged horizontally
  • row-reverse: Items arranged in reverse horizontal order
  • column: Items arranged vertically
  • column-reverse: Items arranged in reverse vertical order

2. justify-content

Aligns items along the main axis (horizontal by default).

Common Values:

  • flex-start: Items align to the start
  • center: Items center along the main axis
  • flex-end: Items align to the end
  • space-between: Equal spacing between items
  • space-around: Equal spacing around items
  • space-evenly: Equal spacing between and around

3. align-items

Aligns items along the cross axis (vertical by default).

Common Values:

  • stretch(default): Items stretch to fill the container
  • flex-start: Items align to the top
  • center: Items vertically centered
  • flex-end: Items align to the bottom
  • baseline: Aligns items based on text baseline

4. flex-wrap

Controls whether items stay on one line or wrap to multiple lines.

Values:

  • stretch(default): Items stretch to fill the container
  • nowrap(default): All items on one line
  • wrap: Items wrap onto multiple lines
  • wrap-reverse: Wraps in reverse direction

5. align-content

Aligns a flex containers lines when there is extra space in the cross axis (only applies with multiple lines).

Common Values:

  • flex-start
  • center
  • flex-end
  • stretch(default)
Example Structure:

html

<div class="container">

<div class="item">Box 1</div>

<div class="item">Box 2</div>

<div class="item">Box 3</div>

</div>

CSS

.container {

display: flex;

justify-content: space-between;

align-items: center;

}
Responsive Design

What is Responsive Design?

Responsive Design is a web design approach that ensures a website looks good and functions well on all screen sizes from large desktop monitors to small smartphone screens.

Why is it important?

  • Improves user experience across devices
  • Increases mobile traffic
  • Enhances SEO (Google favors mobile-friendly sites)
  • Future-proofs your website for new devices

Key Techniques

1. Media Queries

CSS feature used to apply styles based on screen size.

CSS

@media (max-width: 600px) {

body {

background-color: lightblue;

}

This example changes the background color on screens 600px or smaller.

2. Fluid Layout

Use relative units like %, em, rem, or vw/vh instead of fixed units like px

CSS

.container {

width: 80%; /* instead of 800px */

}

3. Flexible Images

Make images scale with the screen.

CSS

img {

max-width: 100%;

height: auto;

}

4. Mobile-First Design

Make images scale with the screen.

CSS

/* Mobile first */

body {

font-size: 16px;

/* Larger screens */

@media (min-width: 768px){

body {

font-size: 18px;

}

5. Viewport Meta Tag

Ensures the browser knows how to scale the page on mobile devices.

html

<meta name="viewport" content="width=device-width, initial-scale=1.0">
